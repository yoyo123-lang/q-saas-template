## Instrucciones para Claude Code

Crear ADR en `docs/decisions/` como `ADR-0001-nombre-descriptivo.md`.

**Crear ADR cuando**: se elige base de datos/framework/librería importante, cambia la arquitectura, se define autenticación/pagos, hay múltiples opciones razonables.

**NO crear ADR para**: cambios de estilo, bugfixes, dependencias menores.

Los ADRs capturan **decisiones, no implementaciones**.
